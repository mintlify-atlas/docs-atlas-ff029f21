## What is the MetaMask Extension?

The MetaMask browser extension connects users to the Ethereum blockchain and compatible networks. It manages private keys, signs transactions, and injects an `ethereum` provider into web pages — enabling decentralized applications (dapps) to request user accounts and submit transactions.

The extension is built with:
- **React + Redux** for the UI layer
- **TypeScript** for all new code
- **Browserify + LavaMoat** for production builds (supply-chain security)
- **Webpack** for development builds
- **Jest** for unit tests and **Playwright/Selenium** for E2E tests
